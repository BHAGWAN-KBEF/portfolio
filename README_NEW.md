# Portfolio Website

**Copyright (c) 2025 Kyei-Baffour Emmanuel Frimpong. All rights reserved.**

Modern, responsive portfolio website built with Python Flask, featuring:

- **Database-driven content management**
- **Secure admin panel** for managing projects, experience, certifications
- **PostgreSQL support** for production deployment
- **Contact form** with email notifications
- **Responsive design** with dark/light theme
- **SEO optimized** with proper meta tags

## Features

- ✅ **Admin Dashboard** - Complete CRUD operations for all content
- ✅ **Project Management** - Multiple categories, demo links, technologies
- ✅ **Experience Timeline** - Professional background with achievements
- ✅ **Certifications** - Professional credentials display
- ✅ **Skills Management** - Categorized skills with proficiency levels
- ✅ **Contact System** - Form submissions with admin notifications
- ✅ **Security** - CSRF protection, rate limiting, authentication

## Tech Stack

- **Backend**: Python Flask, SQLAlchemy
- **Database**: SQLite (dev) / PostgreSQL (production)
- **Frontend**: Tailwind CSS, JavaScript
- **Security**: Flask-WTF, Flask-Limiter, Werkzeug

## Setup

1. Create virtual environment: `python -m venv venv`
2. Activate: `venv\Scripts\activate` (Windows)
3. Install dependencies: `pip install -r requirements.txt`
4. Run: `python main.py`
5. Access at: `http://127.0.0.1:3000`
